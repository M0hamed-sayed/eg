# آلة حاسبة - Calculator

آلة حاسبة بسيطة وتفاعلية تدعم العمليات الحسابية الأساسية.

## الملفات المتوفرة

### 1. calculator.js (JavaScript/Node.js)
آلة حاسبة تفاعلية باستخدام Node.js - **جاهزة للاستخدام**

### 2. Calculator.java (Java)
آلة حاسبة باستخدام Java - تحتاج إلى تثبيت Java لتشغيلها

## كيفية الاستخدام

### تشغيل النسخة JavaScript:
```bash
node calculator.js
```

### تشغيل النسخة Java (إذا كان Java مثبتاً):
```bash
javac Calculator.java
java Calculator
```

## المميزات

- ✅ الجمع (+)
- ✅ الطرح (-)
- ✅ الضرب (×)
- ✅ القسمة (÷)
- ✅ معالجة الأخطاء (القسمة على صفر)
- ✅ واجهة تفاعلية سهلة الاستخدام
- ✅ دعم اللغة العربية

## مثال على الاستخدام

```
=================================
    مرحباً بك في الآلة الحاسبة    
=================================

اختر العملية:
1. الجمع (+)
2. الطرح (-)
3. الضرب (×)
4. القسمة (÷)
5. الخروج

اختيارك: 1
أدخل الرقم الأول: 10
أدخل الرقم الثاني: 5

---------------------------------
النتيجة: 10 + 5 = 15
---------------------------------
```
